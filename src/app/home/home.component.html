Demonstração de Componentes

Apresentação de componentes possívelmente uteis para desenvolvimento de aplicações, todos os componentes são reutilizaveis usando ng-content ou sendo modificados por json como nnno componente de tabela

Seções

Cada seção tem um estilo de componente

Elementos

São componentes que mostram elementos de posicionamento e carregamento

Coleções

Nessa seção temos componentes que dependem de dados para ser feita como tabelas

Views

São componentes de contagem de visualizações somente amostragem

Modos

São componentes que mostra componentes que apresentam jeitos diferentes de mostrar dados como modais e accordions
